# Live Example

This is the live example pack for people who want to build their own custom Discord profile widget from this repo.

Use this as:

- a ready-to-import sample
- a reference for how the JSON fields map to a profile-style widget
- a starting point to make your own widget for your profile

## Layouts used

- widget_top_hero
- widget_bottom_stats
- add_widget_preview_hero

## What to edit

All visible content is driven by dynamic keys in the matching config example:

- top_title
- top_subtitle_1
- top_subtitle_2
- top_subtitle_3
- stat_1_label through stat_6_label
- stat_1_value through stat_6_value
- top_image_url

Nothing in the template is hard-coded as a final value.
